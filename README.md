TO RUN:
Install npm 
from your root directory move to "backend" directory
cd backend
Run:
    pip install -r requirements.txt
(this will install all the required modules)
    python main.py
(this will start the backend server)
Now move to your "frontend" directory
after installing npm run:
    npm run dev

ABOUT:
The application's local host link will be available 
This project was made in aim to solve the issues of small scale businesses by managing and helping them take descisions financially.
The project has a scoring system for each payment and hence prepares a leaderboard. This can later be modified as per user input too.
